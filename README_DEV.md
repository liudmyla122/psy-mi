# Инструкция по запуску проекта PSY MI

## Автоматический запуск (рекомендуется)

Для запуска frontend и backend серверов одновременно:

```bash
npm run dev
```

Эта команда автоматически запустит:
- **Backend сервер** на порту `3000` (http://localhost:3000)
- **Frontend сервер** на порту `5173` (http://localhost:5173)

Оба сервера будут запущены параллельно, и вы увидите логи обоих в одном окне терминала с цветной маркировкой:
- `SERVER` - логи backend сервера (cyan)
- `CLIENT` - логи frontend сервера (magenta)

## Запуск отдельных серверов

Если нужно запустить только один из серверов:

### Только Frontend:
```bash
npm run dev:client
```

### Только Backend:
```bash
npm run dev:server
```

## Остановка серверов

Нажмите `Ctrl + C` в терминале, чтобы остановить оба сервера.

## Первый запуск

Если вы запускаете проект впервые:

1. Установите зависимости для frontend:
```bash
npm install
```

2. Установите зависимости для backend:
```bash
cd server && npm install && cd ..
```

3. Настройте базу данных (если еще не настроена):
```bash
cd server && npm run prisma:generate && npm run prisma:migrate && cd ..
```

4. Запустите проект:
```bash
npm run dev
```

## Проблемы?

Если возникают проблемы с запуском:

1. Убедитесь, что порты 3000 и 5173 свободны
2. Проверьте, что все зависимости установлены
3. Убедитесь, что база данных настроена и запущена
4. Проверьте файл `server/.env` на наличие необходимых переменных окружения
