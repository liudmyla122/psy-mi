# ⚡ Быстрый деплой на Vercel

## ✅ Проект готов к деплою!

Все ошибки исправлены, сборка проходит успешно.

## 🚀 Пошаговая инструкция (5 минут)

### Шаг 1: Откройте Vercel
Перейдите на **https://vercel.com** и войдите через GitHub

### Шаг 2: Импортируйте проект
1. Нажмите **"Add New..."** → **"Project"**
2. Найдите репозиторий **`liudmyla122/psy-mi`**
3. Нажмите **"Import"**

### Шаг 3: Настройки (уже настроены автоматически)
- ✅ Framework: **Vite**
- ✅ Build Command: `npm run build`
- ✅ Output Directory: `dist`
- ✅ Install Command: `npm install`

**Ничего менять не нужно!** Просто нажмите **"Deploy"**

### Шаг 4: Дождитесь деплоя
Обычно это занимает 2-3 минуты

### Шаг 5: Получите ссылку
После деплоя вы получите ссылку вида:
```
https://psy-mi.vercel.app
```
или
```
https://psy-mi-liudmyla122.vercel.app
```

## 🎉 Готово!

**Отправьте эту ссылку заказчику** - сайт работает в реальном режиме!

## 📝 Что дальше?

- Каждый раз когда вы делаете `git push`, сайт автоматически обновляется
- Все изменения появляются на сайте через 2-3 минуты после push

## ⚠️ Важно для бэкенда

Если нужен бэкенд (API), создайте **отдельный проект** в Vercel:
- Root Directory: `server`
- Framework: Other
- Build Command: `npm install && npm run build`

Но для просмотра фронтенда это не обязательно!
